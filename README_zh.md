一本协作式的 IPv6 书籍。

本书旨在为技术人员提供 IPv6 的实用介绍，由活跃的从业者持续更新。

本书将免费在线提供（无广告），可能还会提供按成本价获取印刷版的选项。

这个 GitHub 仓库是一个起点。随着工作的进展和我们对如何进行的经验积累，它可能会被一个不同结构的仓库所取代。

你可以从[这里](Contents.md)开始阅读本书。这是一个正在进行的工作，如果你想要贡献，请从[这里](CONTRIBUTING.md)开始。
