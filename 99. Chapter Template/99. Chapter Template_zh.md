# 章节模板

本章展示了如何编写新章节。它被有意地列在本书的目录中，旨在成为活的书的活的章节。在贡献之前，您还应该查看 [CONTRIBUTING](https://github.com/becarpenter/book6/blob/main/CONTRIBUTING.md) 和 [LICENSE](https://github.com/becarpenter/book6/blob/main/LICENSE.md)。

本章的第二节展示了如何编写新小节。

我们使用 GitHub 方言的 Markdown。下面的 [<ins>Markdown 使用</ins>](Markdown%20Usage.md) 小节中有一些相关信息，包括如何包含图表。

章节位于其自己的目录中，例如，本章位于目录 `99. Chapter Template` 中。当然，空格是目录名称的一部分，并且名称是区分大小写的。

前九章从 1 到 9 _编号_，但从 2025 年 1 月开始，它们的目录_命名_为"01"到"09"，以匹配 GitHub 排序目录的方式。

章节的介绍（如本文件）是一个与目录名称相同的 markdown 文件，例如 `99. Chapter Template.md`。

此文件的第一行是：

```
  # Chapter Template
```

因此，这是 `Chapter Template` 的三次重复。

首先为章节编写一般介绍。告诉读者该章节的内容。然后给出小节列表。可以将小节直接嵌入此处，但由多个作者进行维护时，每个小节使用单独的文件会更容易。因此，编写介绍性文本，然后添加您打算编写的小节列表。例如，在介绍之后，放置：

```
  ## First Section
  ## Next Section
  ## Last Section
```

请保持小节名称简短。它们也用作文件名。小节 `Next Section` 的文本将位于名为 `Next Section.md` 的文件中。

请不要在小节列表内部或之后添加文本。这会使事情变得混乱。（但是，对于具有短小节的短章节，可以忽略此请求。如果您执行以下操作：

```
  ## Section A
  A sentence or three.
  ## Section B
  A few more sentences.
```

那么不需要为每个小节创建单独的文件。）

重要提示：Markdown 无法处理带有空格的文件名。创建链接时，我们必须用 %20 替换空格，或避免在文件名中使用空格。因此，插入链接后的小节列表模板如下：

```
  ## [First Section](First%20Section.md)
  ## [Next Section](Next%20Section.md)
  ## [Last Section](Last%20Section.md)
```

这有点复杂，并且由于文件名区分大小写，因此很容易出错。因此，存在一个名为 makeBook 的 Python 程序，可以偶尔运行它以自动创建此类链接，并协调实际章节内容与主 [Contents](../Contents.md) 页面之间的差异。

它还做一些其他事情来帮助作者：

1. 如果它在主目录中找到一个章节，但没有相应的章节目录，它会创建后者，并提供作者可以编辑的适当模板文件。因此，在 Contents.md 的明显位置添加 `25. Interesting Stuff` 会起作用。

1. 如果它在章节介绍中找到一个小节，但没有相应的 Markdown 文件，它会创建这样的文件。作者只需添加内容即可。

1. 如果它在章节目录中找到一个 Markdown 文件，但它不在章节目录中，它会将其添加到目录列表中。

1. 它会自动在每个小节的底部插入指向上一个和下一个小节（如果存在）以及返回到章节介绍的链接。

1. 它会扩展某些引用，如 [Markdown Usage](Markdown%20Usage.md) 中所述。

makeBook 必须时不时地在主 Github 分支上运行。在撰写本文时，对此的实际经验有限 - 请耐心等待。如果出错，没有什么是不能手动修复的。主要规则是：不要弄乱自动生成的链接。

所以，重申一下：向章节介绍添加一个新的 ## 项目，makeBook 将创建必要的 .md 文件。向章节目录添加一个新的 .md 文件，makeBook 将把它添加到章节目录中。

_专业提示：_添加新章节、重命名或删除小节或章节，或将小节从一个章节移动到另一个章节等，目前没有自动化，可能需要大量手动工作。为此，请参阅[特殊说明](https://github.com/becarpenter/book6/blob/main/utilities/chapterReorg.md)。
