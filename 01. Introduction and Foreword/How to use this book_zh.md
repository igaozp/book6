## 如何使用本书

本书是，我们希望将永远是，一个正在进行中的工作。它面向使用互联网协议版本 6（IPv6）规划、部署、维护和运营计算机网络的人员。它正是由这样的人编写和更新的。IPv6 是一个成熟的协议，但每天我们都在获得更多经验，产品在更新，底层技术标准也经常更新。因此，本书也将不断更新。它根据[开源许可证](https://github.com/becarpenter/book6/blob/main/LICENSE.md)发布。欢迎您自费制作印刷副本，但请注意本书将不断演变。

[目录](../Contents.md)应作为所涵盖主题的在线指南。大多数读者可能不会从头到尾阅读。设计您自己通过本书的路径。

还有一个[索引](../Index.md)。

如果您在 GitHub 上阅读本书，这里有一个小提示：出于某种原因，GitHub 不支持在新浏览器标签或窗口中自动打开链接，因此点击链接总是会让您离开当前页面。为了避免这种情况，在大多数浏览器中，您可以使用 CTRL+点击（在 Windows 和 Linux 上）或 CMD+点击（在 MacOS 上）来打开新标签。
