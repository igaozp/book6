## 用户眼中的 IPv6

答案应该是：*他们看不到*。在理想的世界中，用户永远不需要意识到协议栈的底层，当然更不应该看到十六进制数字，甚至不需要意识到他们正在使用 IPv6。网络设计者或运营者的目标应该是让这成为现实。

然而，这不太可能总是成功。如果用户确实看到与 IPv6 相关的内容，很可能是在最糟糕的时候：当出现故障或系统配置问题时。那正是用户要么在阅读在线帮助信息，要么在联系帮助台的时候。因此，建议审查您提供给用户或帮助台工作人员的任何文档，以确保在提到 IPv6 时，信息是完整、正确和最新的。同样重要的是，配置工具的设计应避免或尽量减少用户手动输入 IPv6 地址的需求。

附注：如果您想知道现在是否可以使用 IPv6，请尝试 https://ipv6test.google.com/。本书托管在 GitHub 上，GitHub 对许多功能支持 IPv6，但并非全部。
