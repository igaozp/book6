## 如何贡献

如果您在本书中发现错误或空白，或者根据实际经验不同意某些建议，我们非常欢迎您提出问题，甚至更好的是起草更新或新的文本。我们使用 GitHub 维护本书——请参阅 [book6 仓库](https://github.com/becarpenter/book6)。

您可以通过本书的[问题跟踪器](https://github.com/becarpenter/book6/issues)提出问题。一般性讨论也在 [GitHub 上](https://github.com/becarpenter/book6/discussions)进行。

要成为积极的贡献者，请[查看条件](https://github.com/becarpenter/book6/blob/main/CONTRIBUTING.md)和[说明](https://github.com/becarpenter/book6/blob/main/99.%20Chapter%20Template/99.%20Chapter%20Template.md)。然后提交 GitHub PR。您的贡献将由编辑团队审核。

参与者有一个简短的[行为准则](https://github.com/becarpenter/book6/blob/main/CODE_OF_CONDUCT.md)。
